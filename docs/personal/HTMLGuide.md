# 1. Getting Started
## 1.1 Simple Document
```html
<!DOCTYPE html>  
<html>  
<body>  
  
<h1>My First Heading</h1>  
<p>My first paragraph.</p>  
  
</body>  
</html>
```
## 1.2 Headings
```html
<h1>H1 Heading</h1>
```
